INSTRUCCIONES PARA CREAR ICONOS PWA
====================================

Necesitas crear los siguientes iconos en esta carpeta:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

OPCIONES PARA CREAR ICONOS:
============================

1. HERRAMIENTAS ONLINE (Recomendado):
   - https://realfavicongenerator.net/
   - https://favicon.io/
   - https://www.pwabuilder.com/imageGenerator

2. USAR PWA ASSET GENERATOR (Node.js):
   npm install -g pwa-asset-generator
   pwa-asset-generator tu-logo.png ./assets/icons

3. CREAR MANUALMENTE:
   - Diseña un logo de 512x512px con el emoji 🥠 o una galleta
   - Usa Photoshop, GIMP, Canva o Figma
   - Exporta en los tamaños requeridos

4. PLACEHOLDER TEMPORAL:
   - Puedes usar emojis grandes como placeholder
   - Captura de pantalla del emoji 🥠 en diferentes tamaños
   - O usa un color sólido con el texto "GS" (Galleta Suerte)

DISEÑO SUGERIDO:
================
- Fondo: Dorado (#D4AF37) o degradado dorado-rojo
- Icono: Emoji 🥠 o ilustración de galleta de la suerte
- Estilo: Minimalista, reconocible a pequeña escala
- Formato: PNG con transparencia (opcional)

NOTA: La app funcionará sin iconos, pero no se verá profesional
al instalarla como PWA.
